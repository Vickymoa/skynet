# Termux Python Playground yang Tidak Jelas Arahnya

Selamat datang di repository ini.  
Jika kamu membaca README ini dengan penuh harapan akan menemukan kode yang rapi, struktur profesional, atau dokumentasi yang serius, maka sejak awal perlu disampaikan: harapanmu terlalu tinggi.

Repository ini dibuat bukan karena visi besar, roadmap matang, atau mimpi mengubah dunia. Repository ini lahir karena satu hal sederhana: **iseng membuka Termux dan mengetik sesuatu**.

Dan dari situlah semua kekacauan ini bermula.

---

## Apa Sebenarnya Repo Ini?

Jawaban singkatnya: **tempat eksperimen**.  
Jawaban panjangnya: tempat di mana ide-ide setengah matang diketik, dijalankan, error, diperbaiki, lalu error lagi.

Di dalam repo ini kamu mungkin akan menemukan:
- File Python yang namanya masuk akal tapi isinya tidak
- Kode yang kemarin jalan, hari ini error tanpa sebab
- Script kecil yang dibuat jam 2 pagi karena “kayaknya bisa”
- Percobaan AI sederhana yang belum tentu pintar
- Logika yang ditulis dengan penuh percaya diri, lalu diragukan sendiri 10 menit kemudian

Repo ini bukan etalase karya terbaik. Ini lebih mirip **buku coretan** yang kebetulan disimpan online.

---

## Kenapa Dibuat di Termux?

Karena laptop tidak selalu ada.  
Karena HP selalu di tangan.  
Karena rasa penasaran tidak peduli perangkat apa yang dipakai.

Termux dipilih karena:
- Bisa jalan Python
- Bisa bikin error
- Bisa bikin mikir “kenapa aku milih ini?”

Dan justru itu bagian dari proses belajar. Kalau semuanya mulus, mungkin tidak ada yang benar-benar dipahami.

---

## Apakah Repo Ini Cocok Untuk Belajar?

Iya, tapi dengan catatan.

Repo ini cocok untuk:
- Orang yang belajar pelan-pelan
- Orang yang tidak masalah melihat kode berantakan
- Orang yang paham bahwa belajar = banyak salah

Repo ini **tidak cocok** untuk:
- Pencari tutorial instan
- Pencari hasil cepat
- Orang yang alergi error

Di sini, error bukan musuh. Error adalah teman lama yang sering muncul tanpa diundang.
